# React Big Calendar with Drag and Drop

A simple starter for using react-big-calendar with drag and drop addon.

## My-calendar

This is my private calendar that I use to store my project timeline plans. This is a really simple and straightforward app. The data is stored locally in the client's browser (your browser) with local-storage.

## How to use
- Add event: Double-click at the calendar cell
- Delete event: Double-click at the event title/bar
- Move event: Click and hold and drag it
